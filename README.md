# Online_notepad
Online Notepad is the Command Line Utility, that is accessd using the Command Line

Using that `Online-Notepad` you can perform the following tasks
- Add the Note 
- Delete the Notes
- Display All the Notes

1) Add the note 
```
note "My First Note"
```
2) Display All the Notes
  ```
  note -a
  ```
3) Delete the Node
```
note -d 2
```
